¿Qué es GitHub?

GitHub es una plataforma basada en la nube que permite almacenar, gestionar y compartir proyectos utilizando Git, un sistema de control de versiones. Es ampliamente utilizada por desarrolladores, estudiantes y empresas para trabajar de manera colaborativa.

¿Para qué sirve?

  • Guardar proyectos y código fuente.
  • Controlar cambios realizados en los archivos.
  • Trabajar en equipo de forma colaborativa.
  • Compartir proyectos con la comunidad.
  • Crear portafolios profesionales.

Conceptos básicos de GitHub

  • Repositorio: espacio donde se almacena un proyecto.
  • Commit: registro de cambios realizados.
  • Branch: rama independiente para desarrollar nuevas funciones.
  • Pull Request: propuesta para integrar cambios.
  • Clone: copia local de un repositorio.
